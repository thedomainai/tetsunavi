# テツナビ - ライフイベント×行政手続きAIエージェント

## プロジェクト概要

「引越しします」の一言から、AIが自律的にパーソナライズされた手続きロードマップを作成・管理するサービス。MVPでは **引越し** にフォーカスし、拡張性として他のライフイベント（相続・出産等）にも対応可能なアーキテクチャを示す。

## 解決する課題

### 誰の、どんな課題か

引越しをする全ての日本居住者（年間約700万人）が直面する、20〜30種類の行政・民間手続き。

**核心的ペイン**: 「知らない手続きは検索もできない」問題。
- 既存のチェックリスト（SUUMO等）は静的で、全員同じ内容
- 個人の状況（家族構成、車、ペット、子供の学年等）によって必要な手続きが全く異なる
- 手続きの期限・順序・依存関係が複雑
- 必要書類や管轄窓口が分からず、何度も足を運ぶ

### キラーフレーズ

> **「知らないことは検索できない」— AIエージェントが、あなたの代わりに考えます**

## 技術アーキテクチャ

### マルチエージェント構成（ADK）

```
┌─────────────────────────────────────────┐
│           Root Agent (Orchestrator)       │
│         全体フローの制御・状態管理          │
└──────┬──────┬──────┬──────┬──────┬───────┘
       │      │      │      │      │
  ┌────▼──┐┌──▼───┐┌─▼────┐┌▼─────┐┌▼──────┐
  │Interview││Proce-││Docu- ││Sche- ││Loca-  │
  │ Agent  ││dure  ││ment  ││dule  ││tion   │
  │        ││Agent ││Agent ││Agent ││Agent  │
  │対話で   ││手続き ││書類   ││期限   ││窓口   │
  │状況把握 ││特定   ││準備   ││管理   ││案内   │
  └────────┘└──────┘└──────┘└──────┘└───────┘
```

| エージェント | 役割 | 技術 |
|------------|------|------|
| **Root Agent** | ユーザーとの対話管理、エージェント間の調整 | ADK Orchestrator |
| **Interview Agent** | 対話で引越しの状況を把握（家族構成、引越し先、車/ペット有無等） | Gemini (対話) |
| **Procedure Agent** | 状況に基づき必要な手続きを網羅的に特定 | Vertex AI Search (RAG) |
| **Document Agent** | 各手続きの必要書類・記入例を生成 | Gemini (生成) |
| **Schedule Agent** | 期限・依存関係を考慮したスケジュール作成 | Gemini (推論) |
| **Location Agent** | 管轄の役所・窓口の所在地・営業時間を検索 | Google Maps API + Gemini |

### 技術スタック

| レイヤー | 技術 | 用途 |
|---------|------|------|
| フロントエンド | Next.js (React) | WebアプリUI |
| バックエンド | Python + ADK | マルチエージェント制御 |
| AI | Gemini 2.0 Flash | 対話・推論・生成 |
| 知識ベース | Vertex AI Search | 行政手続き情報のRAG |
| データストア | Firestore | ユーザーデータ、進捗管理 |
| デプロイ | Cloud Run | コンテナデプロイ |
| 外部API | Google Maps API | 窓口所在地検索 |

## MVPスコープ（5日間）

### 含めるもの
- 対話による状況把握（5-6問の質問）
- パーソナライズされた手続きリスト生成（20-30項目）
- 各手続きの詳細（必要書類、管轄窓口、期限）
- タイムラインビュー（いつ何をすべきか）
- 手続き完了チェック機能

### 含めないもの（拡張機能）
- 実際の手続きの代行
- リマインド通知
- 他のライフイベント対応（相続、出産等）
- ユーザーアカウント管理

## 3分デモシナリオ

```
Step 1: 「来月、東京都品川区から大阪市北区に引越しします」と入力
Step 2: AIが5問で状況を把握
  「ご家族構成は？」→ 夫婦+小学生の子供1人
  「お車はお持ちですか？」→ はい
  「ペットはいますか？」→ 猫1匹
Step 3: パーソナライズされた手続きリスト（24項目）が自動生成
Step 4: タイムラインで「いつ何をすべきか」を可視化
Step 5: 手続きの1つをタップ → 必要書類、管轄窓口、最寄駅まで表示
Step 6: 完了チェックで進捗管理
```
